### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、驾驶员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 驾驶员
5. 车辆登记
6. 企业论坛
7. 系统公告
8. 个人中心
9. 我的发布

**驾驶员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人中心
5. 车辆登记管理
6. 维修信息管理
7. 事故信息管理
8. 违章信息管理
9. 车辆运营管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 驾驶员管理
4. 用户管理
5. 车辆登记管理
6. 维修信息管理
7. 事故信息管理
8. 违章信息管理
9. 车辆运营管理
10. 企业论坛
11. 系统管理（系统公告、关于我们、系统简介、轮播图管理）
